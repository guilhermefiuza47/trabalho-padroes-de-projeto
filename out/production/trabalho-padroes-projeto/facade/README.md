## Facade (Estrutural)

**Fonte:** https://refactoring.guru/pt-br/design-patterns/facade

### Sobre o padrão

O padrão Facade evita que a lógica do sistema fique toda conglomerada na classe `Main`. Em vez de obrigar o cliente a gerenciar diretamente as diversas partes menores do código, a fachada organiza essas interações internamente e expõe na `Main` apenas os detalhes essenciais e mais importantes que o cliente realmente precisa saber para executar as funcionalidades.
